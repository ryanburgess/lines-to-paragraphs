Lines to Paragraphs
=============
A JavaScript module that converts new lines to paragraph tags.

## Install
```js
npm install lines-to-paragraphs --save-dev
```
## Use
```js
var paragraphs = require('lines-to-paragraphs');

paragraphs(content);
```
